## Team Role

### 1. Backend Developer
Responsible for building and 
maintaining the sever-side logic,
APIs, and database interactions of
the application.

### 2. Database administrator (DBA)
manages the structure, security, and 
performance of the database to ensure
data integrity and availability.

### 3 Frontend developer
designs and develops the user
interface, ensuring a responsive and 
interactive experience for users.

### 4. Project manager
Coordinates team activities,
timelines, and ensure deliverables
align with the project goals and 
deadlines.

### 5. QA Engineer
tests the application to identify
bugs, ensures quality, and verifies
that features work as expected.

### 6.DevOps Engineer
sets up the CI/CD pipeline, monitors
deployments, and ensures smooth
integration and delivery of code.
